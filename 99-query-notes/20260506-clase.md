# Notas de clase 20260506

Podemos revisar que índices están definidos en nuestra base SQLite usando:

```
.indexes
```

Para ver la descripción que tiene un índice se puede usar la función `index_info`:

```sql
PRAGMA index_info(sqlite_autoindex_Species_1);

PRAGMA index_info(sqlite_autoindex_Plots_1);
```

en nuestro caso se tienen sólo los índices automáticos que ayudan a las funciones de inserción debido a 
las restricciones

## Agregación


```sql
SELECT 
    species,
    COUNT(*) AS n_obs
FROM 
    Surveys
WHERE 
    species IS NOT NULL
GROUP BY 
    species
HAVING 
    n_obs > 10
ORDER BY 
    n_obs DESC;
```

## Consultas complejas

```sql
SELECT
    species
FROM (
    -- Top 10 de especies observadas
    SELECT 
        species, 
        COUNT(*) AS n_obs 
    FROM Surveys
    WHERE 
        species IS NOT NULL
    GROUP BY species 
    ORDER BY n_obs DESC 
    LIMIT 10
);
```

```sql
SELECT 
    speciescode,
    scientificname
FROM Species
WHERE speciescode IN (
    SELECT
        species
    FROM (
        -- Top 10 de especies observadas
        SELECT 
            species, 
            COUNT(*) AS n_obs 
        FROM Surveys 
        WHERE species IS NOT NULL
        GROUP BY species 
        ORDER BY n_obs DESC 
        LIMIT 10
    )
);
```

```sql
WITH top_obs_sps AS (
    SELECT
        species
    FROM (
        SELECT species, COUNT(*) AS n_obs FROM SURVEYS GROUP BY species ORDER BY n_obs DESC LIMIT 10
        )
)
SELECT 
    speciescode,
    scientificname
FROM Species
WHERE
    speciescode in top_obs_sps;
```
```sql
WITH sps_counts_top10 AS (
    SELECT species, COUNT(*) AS n_obs 
    FROM Surveys 
    GROUP BY species 
    ORDER BY n_obs DESC 
    LIMIT 10   
),
top_obs_sps AS (
    SELECT species 
    FROM sps_counts_top10
)
SELECT 
    speciescode,
    scientificname
FROM Species
WHERE
    speciescode IN top_obs_sps;
```

## JOIN (unir)

```sql
SELECT 
    scientificname, 
    speciescode,
    COUNT(Surveys.species) as n_obs 
FROM 
    Species 
    INNER JOIN Surveys ON Species.speciescode = Surveys.species
GROUP BY Species.speciescode, scientificname
ORDER BY scientificname DESC;
```

```sql
SELECT 
    scientificname, 
    speciescode,
    COUNT(Surveys.species) as n_obs 
FROM 
    Species 
    LEFT OUTER JOIN Surveys ON Species.speciescode = Surveys.species
GROUP BY Species.speciescode, scientificname
ORDER BY scientificname DESC;
```
